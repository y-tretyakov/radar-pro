# Radar Pro --- D1 Complete Schema

## Purpose

D1 stores normalized entities, computed features, metrics, signals,
insights and projections.

## Core tables

-   datasets
-   dataset_manifests
-   repositories
-   owners
-   feature_definitions
-   feature_values
-   metrics
-   signals
-   insights
-   projections

## Design principles

-   immutable datasets
-   versioned algorithms
-   reproducible calculations
-   indexed hot paths
